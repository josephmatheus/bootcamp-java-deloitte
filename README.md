# Cadastro de Clientes – CRUD

Este projeto foi desenvolvido como parte de um exercício do bootcamp Java para a criação de um CRUD (Create, Read, Update e Delete).
A aplicação simula um cadastro de clientes de uma empresa de consultoria, permitindo o controle dos clientes e do pagamento da mensalidade do plano contratado. Com isso, a empresa pode saber se o cliente está apto a receber a consultoria no mês vigente.

### Objetivo do Exercício

Praticar a implementação das operações básicas de um CRUD.

### Funcionalidades
- Cadastrar clientes
- Listar clientes cadastrados
- Atualizar dados do cliente
- Remover clientes
- Controlar status de pagamento da mensalidade
